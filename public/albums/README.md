# Albums Folder

Place your photo albums here in numbered folders:

```
albums/
├── Album1/
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── ...
├── Album2/
│   ├── photo1.jpg
│   └── ...
└── Album3/
    └── ...
```

The folder names (Album1, Album2, etc.) will be linked to album records in the database.
Album descriptions and metadata come from the database tables.
